最小质因数

Time limit : 4sec / Memory limit : 512MB

Description

***精通数数，它想让你帮他算出[1,n]中所有合数的最小质因数的k次方和。

Input

一行两个整数n,k。

Output

输出一行一个整数表示答案。对 2^64 取模。

Sample Input

10 2

Sample Output

25

以上原題

題目大意

如題面描述

題解

有兩個很自然的暴力：篩法和容斥

我們可以注意到容斥可以做小質數大範圍的情況，而篩法可以做大範圍大質數的情況

那麼我們設一個閾值，質數小時用容斥，質數大時用篩法，大力剪枝即可通過此題

關於剪枝，我們首先根據題意發現質數只需篩到根號級別。容斥時記憶化可以極大提高效率。還有因為質數互不交叉，所以枚舉時先提出當前枚舉數縮小範圍。應該能過了。
